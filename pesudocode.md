# SimpleStudyPlanner – Pseudocode

> โครงร่างโปรแกรมตัวช่วยวางแผนอ่านหนังสือ 1 สัปดาห์ พร้อมตรรกะแต่ละขั้นตอน

## ภาพรวมการทำงาน
- รับเวลาว่างรายวันจากผู้ใช้ และข้อมูล 5 วิชา (ความยาก, ความพร้อม)
- แปลงข้อมูลเป็นน้ำหนักและชั่วโมงที่แนะนำต่อวิชา
- กระจายชั่วโมงลงในแต่ละวันตามเวลาว่าง
- แสดงตารางอ่านและชั่วโมงรวมรายวิชา

## โครงสร้างหลัก
```pseudocode
CLASS SimpleStudyPlanner

    INIT:
        เก็บรายชื่อวิชา 5 วิชา
        สร้างตัวเก็บข้อมูลความยาก ความพร้อม น้ำหนัก และชั่วโมงที่เหมาะสม
        กำหนดเวลาว่างต่อวันเป็น 0 ทั้ง 7 วัน
        เตรียมตารางอ่านหนึ่งสัปดาห์
        เก็บชื่อวันและตำแหน่งของวัน

    FUNCTION format_hours(hours):
        แปลงจำนวนชั่วโมง (ทศนิยม) เป็นข้อความ:
            - ชั่วโมง + นาที
            - หรือ นาทีอย่างเดียว
            - หรือ "0 นาที"
        RETURN ข้อความที่จัดรูปแบบแล้ว

    FUNCTION input_daily_hours():
        วนรับข้อมูลจากผู้ใช้:
            - ถ้าผู้ใช้พิมพ์ "done" ให้หยุด
            - รับชื่อวันและจำนวนชั่วโมงว่าง
            - ถ้าชื่อวันไม่ถูกต้องหรือชั่วโมงไม่อยู่ในช่วง 0–24 ให้แจ้ง error
            - บันทึกเวลาว่างลงในตำแหน่งของวันนั้น
        แสดงสรุปเวลาว่างของทั้งสัปดาห์

    FUNCTION input_subject_info():
        สำหรับทุกวิชา:
            รับคะแนนความยาก (1–5) และความพร้อม (1–5)
            ถ้าคะแนนไม่อยู่ในช่วง ให้แจ้ง error
            บันทึกคะแนนของวิชานั้น

    FUNCTION compute_hours():
        คำนวณเวลาว่างรวมทั้งสัปดาห์

        สำหรับทุกวิชา:
            คำนวณน้ำหนัก = ความยาก × (6 - ความพร้อม)
            ถ้าน้ำหนัก < 1 ให้ตั้งเป็น 1

        คำนวณผลรวมของน้ำหนักทั้งหมด

        สำหรับทุกวิชา:
            คำนวณชั่วโมงที่เหมาะสมของวิชานั้น
            = เวลาว่างรวม × (น้ำหนักรายวิชา / น้ำหนักรวม)

    FUNCTION distribute():
        remaining = ชั่วโมงที่ยังต้องจัดให้แต่ละวิชา

        สำหรับทุกวันในสัปดาห์:
            capacity = เวลาว่างของวันนั้น
            ถ้า capacity = 0 ให้ข้าม

            สำหรับทุกวิชา:
                ถ้าวิชานั้นจัดครบแล้ว ให้ข้าม

                assign = min(remaining[วิชา], capacity)
                บันทึก assign ลงในตารางของวันนี้
                หัก assign ออกจาก remaining และ capacity

                ถ้า capacity หมด ให้หยุดพิจารณาวิชาและไปวันถัดไป

    FUNCTION print_schedule():
        สำหรับทุกวัน:
            แสดงชื่อวัน
            ถ้าไม่มีวิชา ให้แสดงว่าไม่มีการอ่าน
            ถ้ามี:
                แสดงชื่อวิชาและจำนวนชั่วโมงในรูปแบบอ่านง่าย

        แสดงชั่วโมงรวมต่อวิชาแต่ละตัว

    FUNCTION run():
        แสดงชื่อโปรแกรม
        เรียก input_daily_hours
        เรียก input_subject_info
        เรียก compute_hours
        เรียก distribute
        เรียก print_schedule

MAIN:
    สร้าง SimpleStudyPlanner และเรียก run()
```
